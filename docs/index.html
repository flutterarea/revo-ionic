Installing Ionic

Ionic apps are created and developed primarily through the Ionic command line utility (the “CLI”), and use Cordova to build and deploy as a native app. This means we need to install a few utilities to get developing.

Ionic CLI and Cordova

To create Ionic projects, you’ll need to install the latest version of the CLI and Cordova. Before you do that, you’ll need a recent version of Node.js. Download the installer for Node.js 6 or greater and then proceed to install the Ionic CLI and Cordova for native app development:

npm install -g ionic cordova You may need to add “sudo” in front of these commands to install the utilities globally

Once that’s done, you can create your first Ionic app:

ionic start myApp sidemenu

To preview your app in the browser, first navigate to app browser and serve:

cd myApp ionic serve

Platform Guides

For those building native apps for iOS and Android (most of you!), each platform has certain features and installation requirements before you can get the most out of your Ionic and Cordova development.

For iOS developers, take a look at the Cordova iOS Platform Guide and follow the instructions to install or upgrade Xcode, and possibly register for a developer account to start building apps for iOS.

For Android developers, take a look at the Cordova Android Platform Guide and follow the instructions to install the SDK and/or Android Studio to start building apps for Android.

App setup

Now that you have succesfully installed Ionic on your computer and all dependencies, let's start setting the app on your computer.

Create a new app

As we learned earlier, create a new folder on your computer, in your terminal / cmd run

cd

followed by drag and drop the folder you just created. Pressing enter will set the path in your terminal / cmd to the folder. Here you can write

ionic start myApp sidemenu

in order to create your new app. It will be a standard Ionic app with a sidemenu navigation and 2 pages, but you need only the base structure of the app.

Now replace in the app the src folder with the one in the package.

Run the following command to install animations library:

npm install --save css-animator

And now, running

ionic serve

will open the app preview in the browser

App layout elements (HTML)

To modify / add or remove layout elements, you need to be familiar with HTML and CSS techniques. Is not a must to create a simple app based on what we already included in the package, but is good to have a better sense.

Responsive - Grid

The app is 100%, meaning that it will look good on every screen. You can set the layout based on the screen size. If the screen size is changed on the go (portrait to landscape), the elements will auto arrange.

The grid is composed of three units — a grid, row(s) and column(s). Columns will expand to fill their row, and will resize to fit additional columns. It is based on a 12 column layout with different breakpoints based on the screen size.

<ion-grid> <ion-row> <ion-col> 1 of 2 </ion-col> <ion-col> 2 of 2 </ion-col> </ion-row> <ion-row> <ion-col> 1 of 3 </ion-col> <ion-col> 2 of 3 </ion-col> <ion-col> 3 of 3 </ion-col> </ion-row> </ion-grid>

This is a basic grid with auto-width for the columns. Each column will take the same space. You can add no-padding tag to ion-grid if you don't need extra spacing.

If you want to set the width for each column, you can use the 12 columns grid structure and define it like this:

<ion-col col-8>

If you want a column to take full width on small screens, 6 cols on tablet screens and 3 on bigger screens, you can do this:

<ion-col col-12 col-sm-6 col-lg-3>

If you want to have a space of 3 columns before your column,m you can write:

<ion-col col-3 offset-3> <!-- or you can explicit the offset based on the screen size --> <ion-col col-md-3 col-md-offset-3 col-lg-offset-4>

Push and pull if you need to reorder columns based on the screen size:

<ion-col col-md-6 push-md-3>

You can align the content vertically in your columns like this:

<!-- Align at the top --> <ion-row align-items-start> <!-- Align in the middle --> <ion-row align-items-center> <!-- Align at the bottom --> <ion-row align-items-end>

You can align the content horizontally in your columns like this:

<!-- Align at the left --> <ion-row justify-items-start> <!-- Align in the center --> <ion-row justify-items-center> <!-- Align at the right --> <ion-row justify-items-end>

You can always read more about the grid system here: http://ionicframework.com/docs/api/components/grid/Grid/

Side menu

Side menu HTML structure is defined in app/app.html file. The menu elements are located in app/component.ts. Here you will find the list of categories and subcategories, where you can add your own content based on the model.

If you want to have only categories and no subcategories, you can do:

Navigation through pages

You can navigate to a page using a button element too, or even any type of HTML element will do the job.

<!-- HTML --> <button (click)="openPage()"> <!-- This will run a function in .ts file --> <!-- TS --> openPage() { this.navCtrl.push(MyPage) }

If you want to send parameters to the page you are navigating to, you can do:

<!-- TS --> openPage() { this.navCtrl.push(MyPage, { id: 'xyz' }) } <!-- In the final page you navigate to: --> this.customId = navParams.get('id'); // Will be 'xyz'

This behaviour is usefulk if you have a list of items (chat users) and you want to open a single user profile page. You will send the user id as a parameter to the single page.

ion-header

This is your top bar, here you can include a menu button, logo and a few more icon buttons, based on your app needs. Our pages have in general a left menu icon, and the logo to the right side. You can take the home-admin example, where we have a lot more going on in the header.

You can have the open menu function by adding menuToggle tag to any element you want as the trigger.

Background image

If you need to apply a big great looking image in the background of the page, you can apply:

<!-- Static full screen image in the background --> <ion-content style="background: url(assets/img/img-bg.jpg)" class="img-cover"> <!-- Scrollable image on some element background --> <div style="background: url(assets/img/img-bg.jpg)" class="img-cover">

We are using img-cover class because we need the image to scale based on the container dimensions / aspect ratio. This way your image will adapt and fill in the element no matter what.

Next we declare it in our constructor:

constructor(..., public toastCtrl: ToastController) {

Ionic data providers

We have included 3 providers to help you on your way, Wordpress, Woocommerce and Utils provider.

To have a better understanding, providers are like standard functions, you take some variables, submit to a provider and the provider does its job and return a set of results.

If we were to take the Wordpress wp.ts provider, we can find this function:

getRecentPosts(page:number = 1){ return this.http.get( Config.WORDPRESS_REST_API_URL + 'posts?page=' + page) .map(res => res.json()); }

What it does is to take the page number/1 and make a http request to the Wordpress API address, set in config.ts file in the root of src folder of your project. And in return you get the posts from Wordpress website as Json data.
